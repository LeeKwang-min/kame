# Water Sort Puzzle 디자인

## 개요

비커에 섞여 있는 색상별 물을 정렬하는 퍼즐 게임. 레벨 기반 점진적 난이도.

## 핵심 결정사항

| 항목 | 결정 |
|------|------|
| 엔진 | Canvas 2D + 모바일 (`canvas-mobile`) |
| 난이도 | 레벨 기반 점진적 |
| 애니메이션 | 심플 (즉시 이동) |
| 점수 | 도달 레벨 = 점수 |
| 여유 비커 | 빈 비커 2개 고정 |
| 칸 수 | 비커당 4칸 |
| Undo | 무제한 |
| 게임 오버 | 없음 (시간제한 없음, Skip 가능) |

## 레벨 진행

| 레벨 | 색상 수 | 비커 수 (색상 + 빈 2개) |
|------|---------|----------------------|
| 1 | 2 | 4 |
| 2 | 3 | 5 |
| 3 | 4 | 6 |
| 4 | 5 | 7 |
| 5+ | 6~ | 8~ (점진 증가, 최대 제한 설정) |

## 게임 규칙

- 비커의 맨 위 색상만 다른 비커로 옮길 수 있음
- 같은 색상 위에만 부을 수 있음 (또는 빈 비커)
- 연속된 같은 색상은 한 번에 이동
- 모든 비커가 단일 색상으로 정렬되면 레벨 클리어
- 풀 수 있는 퍼즐만 생성 (셔플 알고리즘으로 보장)

## 조작 방식

- 데스크탑: 마우스 클릭으로 비커 선택 → 대상 비커 클릭
- 모바일: 탭으로 동일
- 키보드: S(시작), R(재시작), Z(Undo), N(Skip)

## 퍼즐 생성 알고리즘

풀 수 있는 퍼즐 보장을 위해 역방향 생성:
1. 완성 상태(각 비커에 단일 색상 4칸)에서 시작
2. 랜덤하게 합법적인 이동을 역으로 수행하여 섞기
3. 충분히 섞인 상태를 초기 퍼즐로 사용
