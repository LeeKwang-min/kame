# Tap Empire (탭 제국) - 게임 디자인 문서

## 개요
3분 안에 최대한 큰 제국을 건설하는 시간제한 아이들 클리커 게임.
탭으로 골드를 획득하고, 자동 생산기를 구매하여 골드 수익을 극대화한다.

## GAME_META
- **id**: `tapempire`
- **title**: 탭 제국
- **engine**: canvas
- **platform**: both
- **touchControls**: tap
- **orientation**: portrait
- **category**: action
- **difficulty**: progressive

## 핵심 게임 메커니즘

### 기본 구조
- 제한 시간: 3분 (180초)
- 목표: 시간 안에 최대한 많은 골드 획득
- 최종 점수: 총 획득 골드 (누적)

### 골드 획득 방법
1. **수동 탭**: 화면 탭/클릭 → 골드 획득 (기본 1골드, 업그레이드 가능)
2. **자동 생산기**: 구매 후 자동으로 초당 골드 생성

### 생산기 시스템 (6단계)

| 단계 | 이름 | 기본 가격 | 초당 생산 | 테마 |
|------|------|----------|----------|------|
| 1 | 광부 | 15 | 1/s | 곡괭이 |
| 2 | 농장 | 100 | 5/s | 밀 |
| 3 | 대장간 | 500 | 20/s | 모루 |
| 4 | 시장 | 2,500 | 100/s | 마차 |
| 5 | 성 | 15,000 | 500/s | 왕관 |
| 6 | 용 | 100,000 | 3,000/s | 용 |

- 같은 생산기를 여러 개 구매 가능
- 구매할 때마다 가격 15% 증가: `기본가격 × 1.15^보유수량`
- 전략: 싼 걸 많이 사 vs 비싼 걸 빨리 사

### 업그레이드 시스템
- 각 생산기마다 효율 2배 업그레이드 (가격: 기본가 × 10)
- 탭 파워 업그레이드: 탭당 골드 2배 (가격 단계적 증가)
- 업그레이드는 1회성

```
┌─────────────────────────┐

│    숫자 파티클 (+5!)     │  탭 시 골드 숫자가 튀어오름

│                         │
├─────────────────────────┤
│ 📊 초당 생산: 126/s      │  생산 요약 바
├─────────────────────────┤
│ ⛏ 광부  x3    [구매 50] │
│ 🌾 농장  x1    [구매 115]│  생산기 목록

│ 🏰 성   🔒           │  잠김 표시 (가격 도달 시 해제)
│ 🐉 용   🔒           │
├─────────────────────────┤
│ [⬆탭파워 200] [⬆효율 500]│  하단: 업그레이드 버튼들
```

### 시각적 피드백
- 탭 시: 골드 숫자가 파티클로 튀어오름 (+1, +5 등)
- 생산기 구매 시: 해당 행이 반짝이는 효과
- 업그레이드 시: 화면 전체 짧은 플래시 효과
- 숫자 롤링: 골드가 증가할 때 슬롯머신처럼 숫자가 굴러감
- 생산기 잠금 해제: 자물쇠가 열리는 애니메이션

### 색상 팔레트
- 배경: 다크 네이비 (#0f1923)
- 골드 텍스트: 골드 (#FFD700)
- 생산기 버튼: 다크 그레이 + 골드 보더
- 구매 가능: 밝은 초록 하이라이트
- 구매 불가: 어두운 회색 (비활성)

## 게임 흐름 & 상태 관리

### 게임 상태 흐름
```
시작 화면 → 로딩 → 플레이 중 → (일시정지 가능) → 게임 오버
                     ↑                                    │
                     └────── 재시작 ──────────────────────┘
```

### 키보드 조작 (데스크탑)

| 키 | 동작 |
|----|------|
| S | 시작 / 재개 |
| P | 일시정지 |
| R | 재시작 |
| Space | 탭 (골드 획득) |
| 1~6 | 해당 번호 생산기 구매 |
| Q | 탭 파워 업그레이드 |
| M | 음소거 토글 |

### 터치 조작 (모바일)
- 상단 탭 영역 탭: 골드 획득
- 생산기 행 탭: 해당 생산기 구매
- 업그레이드 버튼 탭: 업그레이드 구매
- 시작 화면에서 아무 곳이나 탭: 게임 시작

### 사운드
- 탭: 코인 소리 (짧고 경쾌)
- 구매: 차칭 소리
- 업그레이드: 레벨업 사운드
- 게임 오버: 팡파레
